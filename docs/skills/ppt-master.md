# ppt-master

## 快速判断

`ppt-master` 是一个把文档、网页、Markdown 或已有演示素材转换为原生可编辑 PPTX 的工作流。它不是把整页图片塞进 PowerPoint，而是先生成高质量 SVG 页面，再导出为 PowerPoint 中可编辑的文本、形状、图表、图片和动画对象。

## 适合场景

- 你手上有 PDF、DOCX、网页、Markdown、表格、旧 PPT 或长文，需要整理成正式演示稿。
- 你希望最终文件是原生可编辑 PowerPoint，而不是只可播放、不可编辑的图片版 PPT。
- 你需要复用模板、品牌规范、图标、图片素材和固定画布比例。
- 你想保留演讲备注、动画、旁白或更接近 PowerPoint 原生对象的编辑体验。
- 你愿意按工作流逐步生成、预览、检查和迭代，而不是只输入一句话后完全不校对。

## 输入与输出

输入：

- PDF、DOCX、PPTX、URL、Markdown、Excel/CSV、图片素材或文本说明
- 可选模板、品牌规范、画布比例、页数、视觉风格和输出要求

输出：

- 一份原生可编辑 PPTX
- 中间 SVG 页面与导出备份
- 结构化项目目录、来源文件、图片资源和设计规格
- 可选演讲备注、页面动画、旁白音频和实时预览结果

## 核心特色

- 原生可编辑：目标是 PowerPoint 可编辑对象，而不是单纯把幻灯片做成图片。
- 多格式输入：支持把 PDF、DOCX、网页、Markdown、表格和旧 PPT 等素材纳入同一套流程。
- SVG 中间层：先用 SVG 控制页面视觉，再通过脚本导出为 PPTX，兼顾视觉一致性和可编辑性。
- 模板与品牌复用：可以围绕模板、品牌色、字体、图标和页面结构生成演示稿。
- 本地工作流：项目文件、来源素材和导出文件在本地目录中组织，便于追踪和重复导出。
- 实时预览：生成 SVG 阶段可以开启浏览器预览，方便逐页检查布局、文字和图像。
- 备注与动画：导出阶段可保留演讲备注，并支持常见页面转场和元素动画配置。

## 工作流程

1. 接收源材料，并把 PDF、DOCX、网页、表格等转换为可处理的 Markdown 或素材清单。
2. 初始化项目目录，导入来源文件、图片资源和设计约束。
3. 规划演示稿结构、页面节奏、画布比例、字体、颜色、模板和资源使用方式。
4. 按页顺序生成 SVG 页面，通过实时预览检查效果。
5. 对 SVG 做质量检查、图像嵌入、文本处理和 PowerPoint 兼容处理。
6. 导出原生 PPTX，并按需要加入备注、动画或旁白。
7. 根据预览、用户注解或人工校对结果重新调整并再次导出。

## 当前限制

- 复杂内容仍需要明确输入、阶段化执行和人工复核，不适合当作完全自动的“一句话出片”工具。
- 输出质量取决于源文档质量、模型能力、页面设计规格和逐页检查。
- 需要本地 Python 脚本、项目目录和连续工作流，新手应先按原项目 README 完成安装与初始化。
- 财务、法律、合规、密集表格等高精度材料仍需要人工核对文字、数据和图表。
- 长篇或复杂演示稿更适合分批生成、预览和修订。
